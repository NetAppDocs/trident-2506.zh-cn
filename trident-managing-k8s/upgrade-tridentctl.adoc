= 使用 tridentctl 进行升级
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用以下方法轻松升级现有的Trident安装 `tridentctl`。

.关于此任务
卸载并重新安装Trident相当于升级。卸载Trident时， Trident部署使用的持久卷声明 (PVC) 和持久卷 (PV) 不会被删除。在Trident离线期间，已配置的 PV 仍将可用； Trident恢复在线后，将为在此期间创建的任何 PVC 配置卷。

.开始之前
审查link:upgrade-trident.html#select-an-upgrade-method["选择升级方式"]升级前使用 `tridentctl`。

.步骤
. 运行卸载命令 `tridentctl`移除与Trident关联的所有资源，但保留 CRD 和相关对象。
+
[listing]
----
./tridentctl uninstall -n <namespace>
----
. 重新安装Trident。参考link:../trident-get-started/kubernetes-deploy-tridentctl.html["使用 tridentctl 安装Trident"]。



IMPORTANT: 请勿中断升级过程。确保安装程序运行完成。
